回忆录 ​

目前职业：退堂鼓一级鼓手，Web 全干工程师。

为什么会选择 Web 开发这一方向 ​

我选择做 Web 开发也是因为大约 2012 年小升初的时候就接触到了程序相关的知识，那时候 QQ 刷钻非常火，机缘巧合下我成为了一个卡盟的代理，上面除了有刷钻的教程外也有其他的技术，比如生成 QQ 木马、易语言、网站搭建教程，就这样我开始琢磨上面的知识，慢慢的我学会了生成木马（按现在的说法就是脚本小子），用易语言写软件，唯独只有网页我始终搞不定，当时的我手上虽然有 ASP 网页程序但只会点开里面的某个程序，然后它会自动在我本机上部署，我也只能自己点开访问，不会把它搞得让别人也能使用，直到后来我无意间在初中信息技术书上看到了一篇文章，上面就是教你如何把开发完的 HTML 发布到互联网上，我把整本书认认真真的看了一遍，后来在我家对面的网吧二楼阁楼里，朋友约我并带我第一次打英雄联盟，我玩了两把觉得没意思，就准备按照书上的教程操作一遍，我学着书上的在高高兴兴申请一个免费的虚拟主机，学着使用 8U FTP 连接主机的目录，学着写了一个简单的 HTML 页面并上传空间目录，学着用免费提供的三级域名访问，就这样我能够成功的在自己电脑上显示出来，于是我立马把前面提到的 ASP 程序也给上传上去，再次访问也成功了，激动的我马上把链接发给同学让他也试一试，也一样能访问，就这样我学到了虚拟主机、FTP、域名相关的知识。 后面的我整个初中开始接触何种建站程序，做过各种类型的网站， 有论坛类的，当时比较火的程序有：DISCUZ、PHPWIND，前者好像被腾讯收购了，中间停止过一段时间后面又复活了，后者好像被阿里收到倒闭了。 有笑话、作文、盗版小说类的，当时比较火的程序有：DedeCMS、帝国CMS、Wordpress，没想到前面两个程序现在还活着。 有博客类的，当时比较火的程序有：Wordpress、Zblog、Typecho、Emlog。 当然一个都没有做起来过，一侵权基本马上就关闭了，活的最久的只有这个博客了。 就这样到了大学选专业，肯定毫不犹豫的选择了计算机，当时想着就是正规学习 Web 开发，结果老师却教的 C#，后面干脆就自己学的 PHP，结果 PHP 找不到实习，怒转 Java，就这样逐步走上了 Web 开发的路。

  1. 第一个注册的免费顶级域名：shunzi.tk (2012)
  2. 第一个注册的收费顶级域名：gtv8.cn 大约 12 或 13 年注册的，取这个名字是因为当时初中的我特别喜欢看星空卫视的 GTV 频道，MISS、小苍、七叔也都是那时候认识的。
  3. 第一个备案号：鄂ICP备14015454号，备案的域名是 gouleiba.com，大约 14 或 15 年左右，这个名字是当时有部电视剧特别火，里面有个酒吧就叫这个名字，第二个备案号：鄂ICP备17003910号-1，2017 年 03 月 07 日 16:06。
  4. 第一个免费主机商：高高兴兴免费空间 （2012）
  5. 第一个收费主机商：淘宝买的美国地区空间 （2013）
  6. 第一个云服务器：腾讯云，感谢腾讯高校计划，让我 1元/1月 接触到了服务器，学习了 Linux 系统，搭建各种 Web 环境（2016）
  7. 第一个 CDN、对象存储：七牛云（2014-10-18 18:32:24）
  8. 第一个学习动态网页程序语言：PHP
  9. 第一个桌面编程语言：易语言

在写这篇文章的时候，看着当初发的邮件日期、看着当时 15 年被记录下来的网页快照，感觉好像没多久的事情，却不知不觉已经过去快 10 年了，经历过 PC 互联网时代迁移到移动互联网的时代，看着曾经很火的独立网站流量慢慢变小甚至关站，曾经的记忆也随着时光开始变得模糊起来，现在的我虽然也从事着小时候就热爱的 Web 开发，但基本都是低级的 CRUD，大部分时间在写 SQL，早没了当初的感觉，每天过着两点一线混吃等死的日子。20220707
