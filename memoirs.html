博客折腾记 ​

WordPress(2016) -> ZBlog(2017) -> Typecho(2017) -> 自建 PHP(2018) -> 自建 Java(2021) -> Hexo(2023) -> VuePress(2024) -> VitePress(2024)

回头望时才发现自己也挺能折腾的，文章没写多少，程序换了不少，但是吧，从某种角度上来讲，程序的变更也代表着心态的转换，从原来觉得后台管理很方便能随时随地进后台写文章，再到后来根本不想写后台页面，一来要防止被扫描爆破，二来选一个符合心意的富文本编辑器也是一件令人头疼的事情，恰好当时使用 Typecho 时就开始接触 markdown 写作，于是从自建 Java 开始起就不再写管理页面了，直接采用监听文件夹的方式来读取 markdown 文件并转换为 html 缓存起来，相当于是介于 SSR 和 SSG 之间的一个产物，或者说是热部署的 SSG，后面到了 22 年开始尝试当初理解不了的静态网页生成器，随着时间的流逝，人也在慢慢的改变，从"质疑"、"理解"到"成为"，现在只能说真香，特别是静态页面能部署的地方还多。 是什么导致了这种转变呢？我仔细的想一想，感觉似乎是偏见，一种莫名其妙的抵触心理，用比较流行的话来说就是待在舒适区中不愿意去接触新的东西。 还记得最开始用 markdown 的时候觉得好难用，特别是需要上传图片的时候，那个时候普遍的方案是找个图床平台，然后手动把图片的传上去，之后再把链接粘贴过来，还记得当时最火的图床应该是微博，加载速度快、不限制大小，以至于后来微博被薅的加上了反盗链后一众网站哀鸿遍野，当然也有使用 SMMS、腾讯 COS 等其他的，但不管怎样都是需要手动作业，如果图片需要加上水印、压缩等，步骤会更加的繁琐，直到后面我 19 年左右熟悉 Typora 编辑器后才逐渐能接受全用MD 写作，这完全得力于这个编辑器可以配置图片自动上传功能，并且默认对接了一些平台或程序的 API，这样使得在把图片直接粘贴进去或者右键上传时，其会自动的把图上传到图床然后再把链接替换为对应的地址，这样其操作逻辑可以变得和大多数主流富文本编辑器保持一致了，甚至对于程序员来说甚至会更加的方便，因为它还支持自定义程序，像我就自定义实现了自动添加水印、自动压缩、自动上传到图床的功能，到最后操作步骤比单纯的富文本编辑器更简单。 另外的一个原因则是静态站点的熟悉，当时我一直以为用这种方式每次都需要人工的打包生成静态文件，然后在把生成的文件上传到服务器上，感觉好蠢，虽然另外还有一种方式是可以利用 github action 自动生成，但当时的 github 不支持个人创建私有仓库，这意味着文件要全公开，导致有些不想发布的文章也会被迫公开，所以有点接受不了，而现如今我也都了解到这些问题原来也都有对应的解决方案，比如 github 免费支持个人私有仓库了，也了解到有其他各种静态托管平台可以在仓库有异动时自动拉取仓库文件并编译，这些组合起来最后操作步骤就变成只需要写文章，然后 push 到仓库就完事了，而且还自带版本管理，历史追踪功能。

这些也都算的上成长吧，包括当时的我也不会想到若干年后我会写着曾经不喜欢的 Java 语言作为谋生的技术，还记得那是大学最后一个学期，找不到合适的 PHP 工作，了解到 Java 做 Web 很好找工作，只能快速的学着 Java，有孪生兄弟的 C# 作为基础学着倒是不复杂，但就是觉得 Java 语言很别扭，特别是那个注解，理解不了也接受不了，每天我都在破口大骂哪个XX设计的这玩意，若干年后变成了最熟悉的语言。 自建 PHP 网页快照 Zblog 网页快照

为什么会选择 Web 开发这一方向 ​

我选择做 Web 开发也是因为大约 2012 年小升初的时候就接触到了程序相关的知识，那时候 QQ 刷钻非常火，机缘巧合下我成为了一个卡盟的代理，上面除了有刷钻的教程外也有其他的技术，比如生成 QQ 木马、易语言、网站搭建教程，就这样我开始琢磨上面的知识，慢慢的我学会了生成木马（按现在的说法就是脚本小子），用易语言写软件，唯独只有网页我始终搞不定，当时的我手上虽然有 ASP 网页程序但只会点开里面的某个程序，然后它会自动在我本机上部署，我也只能自己点开访问，不会把它搞得让别人也能使用，直到后来我无意间在初中信息技术书上看到了一篇文章，上面就是教你如何把开发完的 HTML 发布到互联网上，我把整本书认认真真的看了一遍，后来在我家对面的网吧二楼阁楼里，朋友约我并带我第一次打英雄联盟，我玩了两把觉得没意思，就准备按照书上的教程操作一遍，我学着书上的在高高兴兴申请一个免费的虚拟主机，学着使用 8U FTP 连接主机的目录，学着写了一个简单的 HTML 页面并上传空间目录，学着用免费提供的三级域名访问，就这样我能够成功的在自己电脑上显示出来，于是我立马把前面提到的 ASP 程序也给上传上去，再次访问也成功了，激动的我马上把链接发给同学让他也试一试，也一样能访问，就这样我学到了虚拟主机、FTP、域名相关的知识。 后面的我整个初中开始接触何种建站程序，做过各种类型的网站， 有论坛类的，当时比较火的程序有：DISCUZ、PHPWIND，前者好像被腾讯收购了，中间停止过一段时间后面又复活了，后者好像被阿里收购倒闭了。 有笑话、作文、盗版小说、盗版电影类的，当时比较火的程序有：DedeCMS、帝国CMS、Wordpress、飞飞CMS 配合采集器自动更新，真没想到前面两个程序现在还活着。 有博客类的，当时比较火的程序有：Wordpress、Zblog、Typecho、Emlog。 当时基本就是做各种采集站，垃圾站，挂第三方广告赚钱，缺点就是一侵权基本马上就关闭了，要说做的最大的算是小说站，百度权重做到了 3 还是 4，不少 SEO 知识也是做排名时学的，而活的最久的也只有这个博客了。 就这样到了大学选专业，肯定毫不犹豫的选择了计算机，当时想着就是正规学习 Web 开发，结果老师却教的 C#，后面干脆就自己学的 PHP，结果 PHP 找不到实习，怒转 Java，就这样逐步走上了 Web 开发的路。
