H5实现简单的拖拽上传 ​

前段时间准备写图床系统的时候，需要用到拖拽上传的功能，原来看着别人写的时候，觉得挺简单的，自己动手写发现并不简单，还琢磨的好久。特此记录，以免又忘记！

H5实现最简单的拖拽上传的事件有四个：

dragenter：当拖拽元素进入目标元素时触发。

dragover：当拖拽元素在目标元素上移动时触发（必须）。

dragleave：当拖拽元素移除目标元素时触发

drop：当拖拽元素释放时触发（必须）。

其中dragover、drop是必须的，且必须执行event.preventDefault()函数来取消浏览器的默认行为！

然后在dataTransfer.files属性中可以得到拖拽的文件参数，但仅仅得到拖拽文件的参数还不够，还要将文

件读取成需要的对应编码才行，这个时候就需要 new FileReader() 对象（具体属性看

https://developer.mozilla.org/zh-CN/docs/Web/API/FileReader）。转换完成后，只需要给

FileReader对象绑定一个onload事件，该事件在读取操作完成时触发。就可以用ajax来异步上传图片了。

html
<!DOCTYPE html>
<html>
<head>
<title></title>
<meta charset="utf-8">
<style type="text/css">
    #div1{
    background-color: red;
    width: 200px;
    height: 200px;
    }
</style>
</head>
<body>
    <div id="div1"></div>
    <ul id='ul'></ul>
    <script type="text/javascript">
        var div = document.getElementById('div1');
        var ul = document.getElementById('ul');
        div.addEventListener('dragenter', function (event) {
            div.innerHTML = '给我';
        }, false);
        div.addEventListener('dragover', function (event) {
            //取消浏览器默认行为
            if (event.preventDefault)event.preventDefault();else event.returnValue = false;
            div.innerHTML = '放手吧';
        }, false);
        div.addEventListener('dragleave', function (event) {
            div.innerHTML = '别走啊';
        }, false);
        div.addEventListener('drop', function (event) {
            //取消浏览器默认行为
            if (event.preventDefault)event.preventDefault();else event.returnValue = false;
            div.innerHTML = '嗯嗯';
            //得到拖拽文件
            var files = event.dataTransfer.files;
            //循环得到所有拖拽文件实现多文件上传
            for (var i = files.length - 1; i >= 0; i--) {
                // new FileReader对象来实践异步读取文件数据
                var fr = new FileReader();
                // 将拖拽文件读取成 base64编码的形式 方便上传
                fr.readAsDataURL(event.dataTransfer.files.item(i));
                // 绑定load事件，当完成读取时触发
                fr.addEventListener('load', function(){
                    //判断是否为图片
                    if(this.result.indexOf("image/") == -1)
                    {
                    alert('有文件不是图片，或图片已损坏');
                    return false;
            }
            //实现ajax发送数据
            sendData(this.result);
        });
        }
        }, false);
    </script>
</body>
</html>
