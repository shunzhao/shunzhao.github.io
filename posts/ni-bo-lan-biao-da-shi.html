波兰表达式(前缀表达式) ​

波兰表达式是指不用()括号就可以表达计算公式的运算顺序，例如在计算 1+2 * 3 时，为了让1+2先计算，通常情况下是会将1+2用括号括起来，以提高1+2的优先级。

在波兰表达式中，通过将运算符号放在操作数的前面，以达到无需括号即可表达例如上面的(1+2)*3，

转换为波兰表达式则是：* 3 + 1 2，（其实就是运算符放在了操作数的前面，因此也叫前缀表达式）

而它的运算规则也有所不同，它是通过从右往左进行遍历，如果遇到操作数则直接入栈，如果遇到操作符则弹出栈顶的两个元素进行计算，然后将结果重新入栈，直到遍历完为止。

举例说明，计算 * 3 + 1 2

  1. 声明一个临时结果栈
  2. 从右往左遍历，得到 2，1 直接入栈
  3. 再次遍历到：+ 弹出栈顶 1， 2 计算为 3 入栈
  4. 此时栈中为 3
  5. 再次遍历到：3 直接入栈
  6. 再次遍历到：* 弹出栈顶 3， 3 计算为 9 入栈
  7. 此时栈中为 9
  8. 遍历完，结果为栈顶元素 9

中缀表达式 ​

对于生活中常见的 (1+2) * 3，操作符在两个操作数的中间，因此也叫中缀表达式

后缀表达式(逆波兰表达式) ​

有了前面的讲解，后缀表达式顾名思义就是操作符在两个操作数的后边，这种表达法非常适合计算机来使用。

例如中缀表达式：1 + 2 - 3 * (4 + 5) / 7 转换后则是：1 2 + 3 4 5 + * 7 / -

后缀表达式的运算规则是从左往右遍历，遇到操作数是则push到栈中，如果遇到操作符则弹出两个栈顶的元素，计算后再次push到栈中，最后遍历完后，栈顶则元素则是计算的结果。

举例：

  1. 声明一个临时结果栈
  2. 首先遍历到 1和2 直接入栈
  3. 遇到 + ，弹出栈顶元素1和2，计算为 3 然后入栈
  4. 遇到 3，4，5 直接入栈 ，此时栈中有 3，3，4，5
  5. 遇到 + ，弹出栈顶元素4和5， 计算为 9 然后入栈，此时栈中有 3，3，9
  6. 遇到 * ，弹出栈顶元素3和9， 计算为 27 然后入栈，此时栈中有 3，27
  7. 遇到 7 直接入栈 ，此时栈中有 3，27，7
  8. 遇到 / ，弹出栈顶元素27和7， 计算为 3 然后入栈，此时栈中有 3，3
  9. 遇到 -，弹出栈顶元素3和3， 计算为 0 然后入栈，此时栈中有 0
  10. 遍历完，栈顶为 0，结果 为 0

中缀表达式转后缀表达式 ​

java
public static void main(String[] args) {
    // 中缀表达式
    String zhongzhui = "1 + 2 - 3 * (4 + 5) / 7";
    // 临时栈
    Deque<String> stack = new LinkedList<>();
    // 转换后缀表达式
    Queue<String> output = new LinkedList<>();
    // 优先级
    Map<String, Integer> priority = new HashMap<>(6);
    priority.put("+", 1);
    priority.put("-", 1);
    priority.put("*", 2);
    priority.put("/", 2);
    priority.put("(", 9);
    priority.put(")", 9);

    List<String> strings = toSuffixExpression(zhongzhui.toCharArray());
    for (String temp : strings) {
        switch (temp.trim()) {
            case "+":
            case "-":
            case "*":
            case "/":
            case "(":
            case ")":
                // 遇到操作符，先获取栈顶元素，判断其是否优先级是最大的，如果是则压入栈中，需要压入栈中，每次压入栈时，都需要判断里面的优先级是否是同级或者是最低的
                // 当前字符为运算符
                String top = stack.peek();
                // 判断栈顶元素不为null，并且不为(
                if (null != top && !"(".equals(temp)) {
                    // 栈顶元素优先级
                    Integer p1 = priority.get(top);
                    // 当前元素优先级
                    Integer p2 = priority.get(temp);
                    // 判断当前运算符的优先级是否小于等于栈顶运算符，如果是则移入到output队列中，直到栈顶运算符大于当前运算符或栈空为止
                    // 说的简单点可以理解为从栈顶开始把所有的优先级小于等于当前运算符全部移到output队列中，最后再把当前运算符入栈
                    while (null != p1 && !"(".equals(stack.peek()) && p2 <= p1) {
                        output.add(stack.pop());
                        p1 = priority.get(stack.peek());
                    }
                    if (!")".equals(temp)) {
                        stack.push(temp);
                    } else {
                        while (!"(".equals(stack.peek())) {
                            output.add(stack.pop());
                        }
                        stack.pop();
                    }
                } else {
                    // 栈顶元素为null说明栈是空的直接push就行了
                    stack.push(temp);
                }
                break;
            default:
                // 当前字符是操作数，直接output后缀表达式队列
                if (isNumber(temp)) {
                    output.add(temp);
                } else {
                    throw new IllegalArgumentException("非法操作数");
                }
        }
    }
    // 遍历完了 还有的就需要依次pop出栈入output了
    while (null != stack.peek()) {
        output.add(stack.pop());
    }
    System.out.println("output:" + output);
    analysis(output);
}

private static boolean isNumber(String temp) {
    for (char t : temp.toCharArray()) {
        if (!Character.isDigit(t)) {
            return false;
        }
    }
    return true;
}

private static List<String> toSuffixExpression(char[] temp) {
    List<String> z = new ArrayList<>();
    StringBuilder t = new StringBuilder();
    if (null == temp || temp.length <= 0) {
        throw new IllegalArgumentException("参数错误");
    } else {

        for (char c : temp) {
            // 判断是不是数字
            if (Character.isDigit(c)) {
                t.append(c);
            } else {
                if (c != ' ') {
                    if (t.length() > 0) {
                        z.add(t.toString());
                    }
                    z.add(String.valueOf(c).trim());
                    t = new StringBuilder();
                }
            }
        }
    }
    if (t.length() > 0) {
        z.add(t.toString());
    }
    return z;
}

后缀表达式计算代码

java

public static void analysis(Queue<String> output){
    Deque<String> result = new LinkedList<>();
    String temp = null;
    // 从左往右进行遍历，如果遇到两个操作数后面跟着一个操作符时，则计算，然后将其放入到结果栈中
    // 详细算法：
    // 遍历后缀表达式队列，直到结果栈为空
    while (null != (temp = output.poll())) {
        // 如果是操作符 则弹出 结果栈 中栈顶的两个操作数进行计算
        if ("+".equals(temp) ||
                "-".equals(temp) ||
                "*".equals(temp) ||
                "/".equals(temp)) {
            String o1 = result.pop();
            String o2 = result.pop();
            int integer1 = Integer.parseInt(o1);
            int integer2 = Integer.parseInt(o2);
            // 计算完毕后再插入到结果栈中
            result.push(String.valueOf(realCompute(integer2, integer1, temp)));
        } else {
            // 操作数直接入栈
            result.push(temp);
        }
    }
    // 最后栈顶元素就是结果了
    System.out.println(result);
    System.out.println(1 + 2 - 3 * (4 + 5) / 7);
}


// 真实计算
public static int realCompute(int s1, int s2, String operation) {
    int i;
    switch (operation) {
        case "+":
            i = s1 + s2;
            break;
        case "-":
            i = s1 - s2;
            break;
        case "*":
            i = s1 * s2;
            break;
        case "/":
            i = s1 / s2;
            break;
        default:
            throw new IllegalArgumentException("不支持的操作符");
    }
    return i;
}
