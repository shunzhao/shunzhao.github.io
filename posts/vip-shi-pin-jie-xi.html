VIP视频解析——C# 更新到项目特别版 ​

今天中午就想着用C#写一个VIP视频解析软件，等晚上吃完了饭就开始写代码。此次呢一共花了四个小时。从18.00写到了22.04。主要是排版改了又改花了这么长的时间，代码部分也花了快半个小时。 主要还是代码记不住，老师讲的也比较快，要不是我之前自学过易语言(和C#一样，都是所见即所得，只不过代码是用中文写的)，还真跟不上老师的速度。现在对我来说唯一的难点就是记不住单词。

软件呢，其实一点技术含量都没有，解析也只是调用的别人提供的API而已，相信只要有一点入门知识的基本上都能写出来！

最后还是直接记上源码：

V1.0版本：

没调用到数据库，直接把接口和接口名写死了。

源码下载：本地下载 2017-03-30

V2.0版本：

把接口写活了，接口的API通过数据库提取，这样我就可以通过修改数据改更新接口。通过数据库适配器对象——SqlDataAdapter来提取接口

源码下载：本地下载 2017-04-01

V2.5版本：

简单的修改了通过数据库拿数据的方法，这次不是用SqlDataAdapter从数据库拿数据了，而是用只进流的方式读取数据，用SqlCommand 创建操作数据库的对象、用SqlDataReader来一行一行读取数据。

重要的是，这次创建了一个DBhelper类，将连接数据库的方法写在了这个类里面，启动窗口只写了个调用方法和查询语句。提升了简洁性。

源码下载：本地下载 2017-04-06

V3.0版本：

这次我将接口数也写到了数据库里面。这样写的好处就是我可以增加和删除接口了，上面的版本我的接口数量还是写死了。也就是说我只能更新接口的API，但是无法修改接口的数量，永远只能是4个。

这次修改后，我就可以通过数据库来添加和删除接口数量的。

ps：本来是想来一次大改动的，早上的专业课老师刚刚把C#的三层架构给我们讲了，我也准备用三层架构来写一遍的，但是老师准备让我们自己做项目。所以我想就做这个VIP视频解析，到时候再做三层架构吧。

源码下载：本地下载 2017-04-12

项目特别版：

最近C# form结束的时候老师让我们写一个项目练练手，和组员讨论了后，我说是在上面的项目文档当中选一个还是用我本来写好的项目，组员果断决定，就用我写好的VIP视频解析。于是我就开始用三层架构开始写。

项目完成时间 花了2-3天吧，剩余的时间就是美化了。

这个项目特别版分为四个模块

登陆模块

功能：

判断用户名是否存在，判断用户名、密码是否正确，判断用户登陆类型。如果登陆为管理员，那么会展示管理页面，否则隐藏管理页面

注册模块

功能：

判断用户名是否存在 如存在 不可被注册 保证用户名的唯一性。

将用户输入的明文密码采用MD5加密方式存储到数据库中

这样即使服务器被入侵导致数据库泄露，那么黑客得到的也不过只是加密后的密码。保证了用户的安全性

解析模块

功能：

将用户提交的视频URL提交到服务器中解析

管理模块

功能：

增加、 删除、 修改、 查询，所有用户 与API 实现更新功能

开发工具和软件环境

Windows 10 操作系统

Visual Studio 2012

SQL Server 2012

软件在 .NET Frameword 4.5环境下完美运行，

所以建议软件运行环境不低于.NET Frameword 4.5

皮肤插件采用 CSkin
